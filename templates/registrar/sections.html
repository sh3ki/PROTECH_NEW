Required Columns and Format

Column Required Format/Description
ID Optional Leave blank for new sections. If provided and exists, row will be skipped.
Section Name Required Name of the section (e.g., Section A, Rizal, Einstein)
Grade Required Existing grade level (e.g., Grade 7, Grade 8)
Room Number Optional Classroom assignment (e.g., Room 101, Lab 2)
Capacity Required Maximum students (positive number, e.g., 40, 35)

Important Notes

  • Existing Records: If the ID column contains an existing section ID, that row will be skipped to prevent duplicates.
  • Grade Validation: The grade must already exist in the system.
  • Uniqueness: Each section name + grade combination must be unique.
  • Capacity: Must be a positive whole number.
